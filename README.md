# 🔍 AI Code Error Finder & Fixer

A developer tool that analyzes your code for errors, bugs, and warnings using AI — and returns a fully fixed version instantly.

## ✨ Features
- Supports 7 programming languages: Python, JavaScript, HTML, CSS, Java, C++, SQL
- Detects errors, warnings, and suggestions
- Shows complete fixed code side by side
- One-click copy of fixed code
- Animated dark theme UI with particle background
- Mobile responsive design

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Groq AI API (Llama 3.3 70B)
- Prompt Engineering (structured JSON output)
- Canvas API (animated background)

## 🚀 How to Use
1. Clone or download the repository
2. Get a free API key from https://console.groq.com
3. Add your key in `script.js` line 1
4. Open `index.html` in your browser
5. Paste your code, select language, click Find & Fix Errors

## 🐛 Error Types
- 🔴 **Error** — Critical bugs that break the code
- 🟡 **Warning** — Issues that may cause problems
- 🔵 **Suggestion** — Improvements for better code quality
